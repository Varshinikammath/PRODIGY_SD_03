# PRODIGY_SD_03 – Contact Management System (Java)

## Description
This project is a Java-based Contact Management System developed as part of the Software Development Internship at Prodigy Infotech.

The program allows users to manage contacts through a simple console interface. Users can add new contacts, view the list of saved contacts, and delete contacts when needed.

## Features
- Add new contacts with name, phone number, and email
- View all saved contacts
- Delete contacts from the list
- Simple and user-friendly console interface
- Stores contacts in memory during program execution

## Technologies Used
- Java
- ArrayList
- Scanner class

## How It Works
1. The program displays a menu with different options
2. The user selects an option (Add, View, Delete, Exit)
3. When adding a contact, the user enters name, phone number, and email
4. The contact is stored in an ArrayList
5. The user can view or delete contacts from the list
6. The program continues until the user selects the exit option

- Strengthened knowledge of loops and conditional statements
